allensdk.ephys.extract_cell_features.analyze_long_squares(sweep_numbers, nwb_file)[source]¶

Extract features specific to long square sweeps. From the list of sweeps, this will identify the rheobase sweep (lowest-amplitude stimulus inducing a spike), a “hero” sweep (lowest-amplitude sweep 30-60pA above rheobase), and a large number of aggregate spike features for the supplied set of sweep numbers.

Parameters:

sweep_numbers: list

List of long square sweep numbers.

nwb_file: string

File name of NWB file.
allensdk.ephys.extract_cell_features.analyze_ramps(sweep_numbers, nwb_file)[source]¶

Extract ramp-specific features from a set of sweeps in an NWB file. These are primarily aggregate spike features.

allensdk.ephys.extract_cell_features.calculate_input_resistance(subthresh_data)[source]¶

Calculate the input resistance of a sweep using the output of a EphysFeatureExtractor instance applied to long square sweeps. This only uses the negative-going squares, and ignores sweeps that exceed -100 pA since some cells start to move from linear in that regime.
